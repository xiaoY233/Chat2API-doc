## 概述

模型列表页面显示所有可用模型的列表。

## 查看模型

从侧边栏导航到 **模型管理** > **模型列表** 查看：

| 列 | 描述 |
|----|------|
| 模型名称 | 模型标识符 |
| 供应商 | 提供该模型的供应商 |
| 所属账户 | 模型关联的账户 |
| 状态 | 在线/离线/未知 |

## 复制模型名称

### 复制选中模型

1. 勾选需要复制的模型
2. 点击 **复制选中** 按钮
3. 模型名称将复制到剪贴板

### 复制全部模型

点击 **复制全部** 按钮复制所有模型名称。

### 复制格式

复制的模型名称使用逗号分隔：

```
DeepSeek-V3.2,GLM-5,Kimi-K2.5,Qwen3-Max
```

<Callout type="info">
  多个模型名称之间使用英文逗号 `,` 隔开。
</Callout>

## API 使用

### 列出模型

```bash
curl http://localhost:8080/v1/models
```

响应：
```json
{
  "object": "list",
  "data": [
    {
      "id": "DeepSeek-V3.2",
      "object": "model",
      "created": 1234567890,
      "owned_by": "deepseek"
    }
  ]
}
```

### 获取模型详情

```bash
curl http://localhost:8080/v1/models/DeepSeek-V3.2
```
