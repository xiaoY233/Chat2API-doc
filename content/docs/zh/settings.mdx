## 概述

设置页面包含 4 个标签页：

| 标签页 | 描述 |
|--------|------|
| **外观** | 主题、语言、侧边栏设置 |
| **通用设置** | 启动、关闭行为、通知、网络代理 |
| **数据** | 日志设置、导入导出、缓存、重置 |
| **安全** | 凭证加密、日志脱敏 |

## 外观

### 主题

| 选项 | 描述 |
|------|------|
| 浅色 | 浅色主题 |
| 深色 | 深色主题 |

<Callout type="info">
  主题设置会影响整个应用界面，包括侧边栏、对话框和内容区域。
</Callout>

### 语言

| 语言 | 代码 |
|------|------|
| 简体中文 | `zh-CN` |
| English | `en-US` |

切换语言后立即生效，无需重启应用。

### 侧边栏

| 设置 | 描述 | 默认值 |
|------|------|--------|
| 启动时折叠 | 启动时默认折叠侧边栏 | 关 |

## 通用设置

### 启动选项

| 设置 | 描述 | 默认值 |
|------|------|--------|
| 开机自启 | 系统启动时启动应用 | 关 |
| 自动启动代理 | 应用启动时启动代理 | 关 |

<Callout type="info">
  自动启动代理需要开启开机自启才能保证一致性。
</Callout>

### 关闭行为

| 设置 | 描述 | 默认值 |
|------|------|--------|
| 最小化到托盘 | 最小化到系统托盘而非关闭 | 开 |
| 关闭行为 | 关闭窗口时的行为 | 最小化 |

**关闭行为选项：**

| 选项 | 描述 |
|------|------|
| 最小化 | 最小化到系统托盘 |
| 关闭 | 直接关闭应用 |
| 询问 | 每次询问用户 |

### 通知

| 设置 | 描述 | 默认值 |
|------|------|--------|
| 启用通知 | 显示系统通知 | 开 |

### 网络代理

**OAuth 代理模式：**

配置 OAuth 认证时的代理设置。

| 模式 | 描述 |
|------|------|
| `system` | 使用系统代理（默认） |
| `none` | 禁用代理 |

<Callout type="info">
  如果 OAuth 登录失败，可以尝试切换代理模式。
</Callout>

## 数据

### 日志设置

| 设置 | 描述 | 默认值 | 范围 |
|------|------|--------|------|
| 日志级别 | 日志记录详细程度 | `info` | `debug`, `info`, `warn`, `error` |
| 日志保留天数 | 日志保留天数 | 30 | 1-365 |
| 最大日志数 | 最大日志条数 | 10000 | 100-100000 |

**日志级别说明：**

| 级别 | 描述 |
|------|------|
| `debug` | 输出所有日志，适合开发调试 |
| `info` | 输出常规信息及以上级别日志 |
| `warn` | 仅输出警告和错误日志 |
| `error` | 仅输出错误日志 |

### 数据管理

| 操作 | 描述 |
|------|------|
| 导出配置 | 将当前设置导出为 JSON 文件 |
| 导入配置 | 从 JSON 文件导入设置 |

**导出格式：**

```json
{
  "version": "1.0.0",
  "exportedAt": "2026-03-10T...",
  "settings": "..."
}
```

<Callout type="warning">
  导入配置会替换当前设置并刷新页面。
</Callout>

### 危险区域

| 操作 | 描述 | 确认机制 |
|------|------|----------|
| 清除缓存 | 清除会话缓存 | 无确认 |
| 重置应用 | 重置所有数据为默认值 | 对话框确认 |

<Callout type="error">
  重置应用将删除所有数据，包括账户、配置和日志。此操作无法撤销。
</Callout>

## 安全

### 凭证加密

| 设置 | 描述 | 默认值 |
|------|------|--------|
| 启用加密 | 加密存储的凭证 | 开 |

<Callout type="info">
  凭证（令牌、密码）在存储前会使用安全算法进行加密。
</Callout>

### 日志脱敏

| 设置 | 描述 | 默认值 |
|------|------|--------|
| 启用脱敏 | 在日志中隐藏敏感数据 | 开 |

**示例：**

```
原始：sk-1234567890abcdef1234567890abcdef
脱敏：sk-1234****cdef
```

<Callout type="info">
  日志脱敏有助于保护 API 密钥和令牌等敏感信息不被记录在日志中。
</Callout>

## 配置文件

设置存储位置：

| 平台 | 位置 |
|------|------|
| Windows | `%APPDATA%\chat2api\config.json` |
| macOS | `~/Library/Application Support/chat2api/config.json` |
| Linux | `~/.config/chat2api/config.json` |

**配置示例：**

```json
{
  "theme": "system",
  "language": "zh-CN",
  "autoStart": false,
  "autoStartProxy": false,
  "minimizeToTray": true,
  "closeBehavior": "minimize",
  "enableNotifications": true,
  "oauthProxyMode": "system",
  "logLevel": "info",
  "logRetentionDays": 30,
  "maxLogs": 10000,
  "credentialEncryption": true,
  "logDesensitization": true
}
```
