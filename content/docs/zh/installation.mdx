## 系统要求

| 平台 | 要求 |
|------|------|
| macOS | macOS 10.15 (Catalina) 或更高版本，Apple Silicon 或 Intel |
| Windows | Windows 10 或更高版本，64 位 |
| Linux | Ubuntu 18.04+、Debian 10+ 或其他现代发行版 |

## 下载

从 [GitHub Releases](https://github.com/xiaoY233/Chat2API/releases) 下载最新版本：

| 平台 | 下载 |
|------|------|
| macOS (Apple Silicon) | `Chat2API-x.x.x-arm64.dmg` |
| macOS (Intel) | `Chat2API-x.x.x-x64.dmg` |
| Windows | `Chat2API-x.x.x-x64-setup.exe` |
| Linux | `Chat2API-x.x.x-x64.AppImage` 或 `.deb` |

## 安装步骤

### macOS

1. 下载适合您架构的 `.dmg` 文件（Apple Silicon 或 Intel）
2. 打开下载的文件
3. 将 Chat2API 拖到应用程序文件夹
4. 从应用程序启动 Chat2API

<Callout type="warning">
  如果出现"App已损坏，无法打开"的错误，请在终端运行以下命令：
  
  ```bash
  sudo xattr -rd com.apple.quarantine "/Applications/Chat2API.app"
  ```
</Callout>

### Windows

1. 下载 `.exe` 安装程序
2. 运行安装程序
3. 按照安装向导操作
4. 从开始菜单或桌面快捷方式启动 Chat2API

<Callout type="info">
  Windows 可能会显示 SmartScreen 警告。点击"更多信息"，然后点击"仍要运行"继续。
</Callout>

### Linux

#### AppImage（推荐）

```bash
# 下载 AppImage
wget https://github.com/xiaoY233/Chat2API/releases/download/v1.0.0/Chat2API-1.0.0-x64.AppImage

# 添加执行权限
chmod +x Chat2API-1.0.0-x64.AppImage

# 运行
./Chat2API-1.0.0-x64.AppImage
```

#### DEB 包（Debian/Ubuntu）

```bash
# 下载 DEB 包
wget https://github.com/xiaoY233/Chat2API/releases/download/v1.0.0/Chat2API-1.0.0-x64.deb

# 安装
sudo dpkg -i Chat2API-1.0.0-x64.deb

# 如需要，修复依赖
sudo apt-get install -f
```

## 从源码构建

### 前置要求

- Node.js 18+
- npm
- Git

### 构建步骤

```bash
# 克隆仓库
git clone https://github.com/xiaoY233/Chat2API.git
cd Chat2API/Chat2API-Manager

# 安装依赖
npm install

# 启动开发环境
npx electron-vite dev 2>&1
```

### 生产构建

```bash
# 构建应用
npm run build

# 构建特定平台
npm run build:mac          # macOS (dmg)
npm run build:win          # Windows (nsis)
npm run build:linux        # Linux (AppImage, deb)
npm run build:all          # 所有平台
```

## 首次启动

安装完成后，首次启动 Chat2API：

1. 应用将在 `~/.chat2api/` 创建数据目录
2. 自动生成默认配置
3. 您可以开始添加供应商并配置代理
