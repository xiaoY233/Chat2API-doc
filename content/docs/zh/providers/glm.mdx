## 支持的模型

| 模型 | 描述 |
|------|------|
| GLM-5 | 最新 GLM 旗舰模型 |

## 功能特性

- GLM-5 旗舰模型
- 深度思考模式
- 联网搜索
- 流式/非流式输出
- 多轮对话支持
- 工具调用支持

## 认证方式

GLM 支持 **OAuth 登录认证** 和 **Refresh Token 认证** 两种方式。

### 方式一：OAuth 登录认证（推荐）

<Steps>
  <Step title="添加供应商">
    在 Chat2API 中点击**添加供应商**，选择 GLM。
  </Step>
  
  <Step title="选择认证方式">
    选择 **OAuth 登录** 方式。
  </Step>
  
  <Step title="浏览器登录">
    点击**开始登录**，系统会自动打开浏览器跳转到智谱清言登录页面。
  </Step>
  
  <Step title="完成登录">
    在浏览器中完成登录后，返回 Chat2API，系统会自动获取认证信息。
  </Step>
</Steps>

### 方式二：手动获取 Token

<Steps>
  <Step title="访问智谱清言">
    前往 [智谱清言](https://chatglm.cn/) 并登录您的账户。
  </Step>
  
  <Step title="打开开发者工具">
    开始任意对话，然后按 `F12` 打开开发者工具。
  </Step>
  
  <Step title="导航到 Application">
    在开发者工具中，转到 **Application** 标签。
  </Step>
  
  <Step title="查找 Cookies">
    在左侧边栏展开 **Cookies**，选择 `https://chatglm.cn`。
  </Step>
  
  <Step title="复制 Token">
    找到名为 `chatglm_refresh_token` 的 Cookie，复制其值。
  </Step>
</Steps>

<Callout type="info">
  Token 存储在 Cookies 中，而不是 Local Storage。
</Callout>

## 添加到 Chat2API

1. 导航到 **供应商** > **添加供应商**
2. 从列表中选择 **GLM**
3. 粘贴您的 `chatglm_refresh_token` 值
4. 点击 **验证** 确认
5. 保存供应商

```json
{
  "model": "GLM-5",
  "messages": [...]
}
```

| 模型名称 | 实际模型 | 描述 |
|----------|----------|------|
