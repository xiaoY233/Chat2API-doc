## 端口配置

代理服务器在指定端口监听传入的 API 请求。

| 设置 | 默认值 | 描述 |
|------|--------|------|
| 端口 | 8080 | 代理服务器的端口号 |

<Callout type="warning">
  低于 1024 的端口需要管理员/超级用户权限。
</Callout>

### 更改端口

1. 导航到 **代理设置** > **基础配置**
2. 输入新的端口号（1-65535）
3. 点击 **保存配置**
4. 重启代理服务器

## 绑定地址

配置代理绑定到哪个网络接口。

| 地址 | 描述 |
|------|------|
| `127.0.0.1` | 仅接受本地连接（默认） |
| `0.0.0.0` | 接受所有接口的连接 |
| 特定 IP | 绑定到特定网络接口 |

<Callout type="info">
  使用 `127.0.0.1` 仅允许本地访问，或使用 `0.0.0.0` 允许网络访问。
</Callout>

## CORS 设置

跨域资源共享（CORS）允许 Web 应用程序向代理发送请求。

### 启用 CORS

1. 切换 **启用 CORS** 选项
2. 配置允许的来源

### CORS 来源

| 值 | 描述 |
|----|------|
| `*` | 允许所有来源 |
| `http://localhost:3000` | 允许特定来源 |
| 多个 | 逗号分隔的来源列表 |

## 配置文件

设置存储在 `~/.chat2api/config.json`：

```json
{
  "proxyPort": 8080,
  "proxyHost": "127.0.0.1"
}
```
