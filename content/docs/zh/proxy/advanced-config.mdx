title: 高级配置
description: 配置代理服务器的高级设置，包括请求超时和重试次数。
---

## 请求配置

### 超时时间

等待供应商响应的最大时间。

| 设置 | 默认值 | 范围 |
|------|--------|------|
| 请求超时 | 60 秒 | 1-300 秒 |

<Callout type="info">
  较长的超时时间适合复杂查询，但会增加等待时间。
</Callout>

### 重试次数

失败时自动重试的次数。

| 设置 | 默认值 | 范围 |
|------|--------|------|
| 重试次数 | 3 | 0-10 |

## 配置文件

设置存储在 `~/.chat2api/config.json`：

```json
{
  "requestTimeout": 60000,
  "retryCount": 3
}
```
