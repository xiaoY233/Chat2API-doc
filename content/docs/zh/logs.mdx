## 概述

日志页面提供：
- 完整的请求历史
- 请求详情和响应
- 错误追踪
- 性能指标

## 查看日志

### 日志列表

从侧边栏导航到 **日志** 查看：

| 列 | 描述 |
|----|------|
| 时间戳 | 请求发生时间 |
| 方法 | HTTP 方法（POST、GET） |
| 路径 | API 端点 |
| 状态 | HTTP 状态码 |
| 耗时 | 请求处理时间 |
| 模型 | 使用的模型 |
| 供应商 | 使用的供应商 |

### 日志详情

点击任意日志条目查看：
- 完整请求体
- 响应体
- 请求头
- 错误信息（如有）
- Token 使用量

## 筛选

### 按时间范围

- 最近 1 小时
- 最近 24 小时
- 最近 7 天
- 自定义范围

### 按状态

| 筛选 | 描述 |
|------|------|
| 成功 (2xx) | 成功的请求 |
| 客户端错误 (4xx) | 客户端错误 |
| 服务器错误 (5xx) | 服务器错误 |

### 按供应商

按特定供应商筛选日志。

### 按模型

按特定模型筛选日志。

## 搜索

搜索日志内容：
- 请求内容
- 响应内容
- 错误信息
- 模型名称

## 日志级别

| 级别 | 描述 |
|------|------|
| 信息 | 正常操作 |
| 警告 | 潜在问题 |
| 错误 | 失败请求 |

## 导出

### 导出选项

1. 点击 **导出** 按钮
2. 选择格式：
   - JSON
   - CSV
3. 选择时间范围
4. 下载文件

## 统计

### 仪表盘指标

日志贡献仪表盘统计：
- 总请求数
- 成功率
- 平均响应时间
- Token 使用量

### 性能分析

使用日志分析：
- 高峰使用时段
- 慢速端点
- 错误模式
- 供应商可靠性

## 日志保留

日志根据保留设置自动清理：

| 设置 | 默认值 |
|------|--------|
| 保留期限 | 7 天 |

在 **设置** > **通用** 中配置。

## 故障排除

### 常见错误

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| 401 未授权 | 无效 API Key | 检查 API Key 配置 |
| 429 请求过多 | 超过速率限制 | 添加更多账户或降低请求频率 |
| 500 内部错误 | 供应商错误 | 检查供应商状态 |
| 503 服务不可用 | 供应商离线 | 检查供应商连接 |

### 调试模式

启用调试日志获取更多详情：

1. 转到 **设置** > **通用**
2. 将日志级别设置为 **调试**
3. 重现问题
4. 检查日志获取详细信息

<Callout type="info">
  调试模式会生成更多日志。故障排除后请禁用。
</Callout>
